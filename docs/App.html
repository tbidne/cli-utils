MonadIO m => MonadStaleBranches (AppT m) Source #

MonadStaleBranches instance for (AppT m) over MonadIO. This means we can encounter exceptions, but

  • We do not want a single exception trying to parse one branch kill the entire program.
  • We do not want to ignore problems entirely.

We opt to define Handler as (ErrOr a), which is an alias for (Either Err a). We collect the results in ResultsWithErrs, which contains a list of errors and two maps, one for merged branches and another for unmerged branches.
